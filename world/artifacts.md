# 伝説の武具・道具

## 概要

- 術式が刻まれた道具のうち、道具自体が精神エネルギーを発生させる特殊なもの
- 発生メカニズムの詳細は[魔術](魔術.md#魔術の規模)を参照
- 以下の精神エネルギー発生ケースに分類される（詳細は魔術.mdの「魔術の規模」を参照）
  - ケースA: 存在強度が極めて高い道具が集合無意識から精神エネルギーを受信
  - ケースB: 製作時の魔術師の精神エネルギーが封入されている
  - ケースC: 道具自体がある種の個体として意思を持つに至った

## 一覧

### エクスカリバー

- **ケース**: A（集合無意識）
- **精神エネルギー源**: 「選定の剣」として万人に知られることによる極めて高い存在強度。集合無意識から精神エネルギーを受信する
- **特性**:
  - 使用者の資質（精神エネルギーの強さと方向性）に応じて出力が変動する
  - 使用者の精神エネルギーと剣が受信する集合無意識の精神エネルギーが相乗する構造
  - 「正統な王」としての認知が集合無意識を増幅するため、使用者の社会的立場も出力に影響する
- **術式**: 使用者の精神エネルギーを増幅する術式が刻まれており、集合無意識からの受信を使用者に還元する

### 三種の神器（草薙剣）

- **ケース**: A + C
- **精神エネルギー源**: 国家レベルの集合無意識（「日本を守護するもの」としての認知）+ 「守護」という極限まで特化した意思
- **特性**:
  - 日本列島という大規模な個体（地球の一部）との同調を補助する
  - 使用者と日本列島の精神エネルギーの波長の差を術式が橋渡しすることで、通常は不可能な同調を実現する
  - 守護に関する魔法の効率が極めて高い一方、攻撃的な用途には適さない
- **術式**: 精神エネルギーの波長を日本列島の精神エネルギーに近づけるシフト機能を持つ

### ミョルニル

- **ケース**: A + C
- **精神エネルギー源**: 「雷神の武器」としての集合無意識 + 「破壊」に極限まで特化した意思
- **特性**:
  - 現化魔法（エネルギー化: 電磁気エネルギー）に特化した武具
  - 術式が精神エネルギーの方向を「電磁気」に最適化するため、雷に関する魔法の効率が破格
  - 「破壊」の意思により、道具自体が精神エネルギーを発生させ、使用者の出力に加算される
- **術式**: 精神エネルギーの方向を電磁気エネルギーに変換する高効率な術式

### ロンギヌスの槍

- **ケース**: C
- **精神エネルギー源**: 「貫く」という極限の意思。製作の経緯と歴史的な使用を通じて、この方向性が極限まで研ぎ澄まされた
- **特性**:
  - 真化魔法（還元）に特化した武具
  - 「貫く」という意思が存在強度を無視する方向で作用し、通常の真化魔法では不可能な高存在強度の対象に対しても還元を行える可能性がある
  - ただし「現界」の精神エネルギーを超えることは不可能であり、ビッグバン由来の物質への完全な還元はできない
- **術式**: 精神エネルギーをマナ還元の触媒として最適化する術式。対象の存在強度への干渉効率が極めて高い

### 賢者の石

- **ケース**: B
- **精神エネルギー源**: 製作者の精神エネルギーが封入されている
- **特性**:
  - 使用者の魔法・魔術の出力をブーストする道具
  - 封入された精神エネルギーは使用により消耗するため、本質的には消耗品
  - ただし封入量が膨大であるため、実質的に数世代にわたって使用可能な場合もある
  - 意図的な製作は極めて困難であり、超希少
- **術式**: 封入された精神エネルギーを使用者の精神エネルギーに加算する術式
