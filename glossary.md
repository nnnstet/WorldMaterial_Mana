# 世界設定 概念索引

全ドキュメントから重要な概念を抽出し、一言解説とリンクを付けたもの。
新規参入者へのガイドおよび全体の整合性チェックに利用する。

執筆時のルールについては別途管理する。

## 1. 世界構造

| 概念 | 一言解説 | 出典 |
|---|---|---|
| **現界** | 人間が存在する3次元空間（宇宙全体）。物理法則は現実世界に酷似 | [世界の法則 - 現界](world/世界の法則.md#現界) |
| **真界** | 現界とは異なる次元。マナが存在する場所 | [世界の法則 - マナ（真界）](world/世界の法則.md#マナ真界) |
| **マナ** | 真界に存在する純粋なエネルギー。あらゆる物理エネルギー・物質に変換可能なポテンシャルそのもの | [世界の法則 - マナ（真界）](world/世界の法則.md#マナ真界) |
| **マナの濃淡** | 真界内でマナに濃い・薄いがある。変化は数千〜数万年単位で極めて緩慢 | [世界の法則 - マナ（真界）](world/世界の法則.md#マナ真界) |
| **マナの総量** | 真界のマナ ≫ 宇宙全体のエネルギー ≫ 惑星単体のエネルギー | [世界の法則 - マナ（真界）](world/世界の法則.md#マナ真界) |
| **マナの暴走** | マナが極端に濃くなり、魂や精神エネルギーなしに現界に染み出す自然災害的現象 | [世界の法則 - マナ（真界）](world/世界の法則.md#マナ真界) |
| **マナは現界に存在しない** | マナをマナのまま取り出すことは不可能。取り出した時点で物理エネルギーか物質に変換される | [世界の法則 - マナ（真界）](world/世界の法則.md#マナ真界) |

## 2. エネルギー保存・瘴気・存在強度

| 概念 | 一言解説 | 出典 |
|---|---|---|
| **エネルギー保存則** | 「真界のマナ＋現界の物質/エネルギー＋瘴気」の総量は常に一定 | [世界の法則 - エネルギー保存則](world/世界の法則.md#エネルギー保存則) |
| **瘴気** | マナから物質/エネルギーを生成した際の副産物。物質と結合してマナに戻す（反物質に類似） | [世界の法則 - 瘴気](world/世界の法則.md#瘴気) |
| **マナ還元** | 瘴気と物質/エネルギーが結合してマナに戻ること。戻るタイミングで真界に帰る | [世界の法則 - マナ還元](world/世界の法則.md#マナ還元) |
| **存在強度** | 物質/エネルギーが精神エネルギーの影響を受けている度合い。高いほど瘴気と反応しにくい | [世界の法則 - 存在強度](world/世界の法則.md#存在強度) |
| **ビッグバン由来物質の存在強度** | 「現界」自体の精神エネルギーで保持。極限に高く、真化魔法による還元は実質不可能 | [世界の法則 - 存在強度](world/世界の法則.md#存在強度) / [魔法 - 真化魔法の制限](world/魔法.md#真化魔法の制限) |
| **瘴気の対称性** | マナから生成された物質/エネルギーと瘴気は常に1対1の関係 | [世界の法則 - エネルギー保存則](world/世界の法則.md#エネルギー保存則) |
| **集合無意識と存在強度** | 万人に知られた物は存在強度が高い（例：聖剣） | [世界の法則 - 存在強度](world/世界の法則.md#存在強度) |
| **変化部分の存在強度** | マナ由来エネルギーが既存物質に作用した場合、変化部分の安定は術者の精神エネルギーの強さに依存 | [世界の法則 - 存在強度](world/世界の法則.md#存在強度) |
| **二次的物理反応の不可逆性** | マナ由来の熱で点火された化学反応などは物理法則に従い不可逆的に維持される | [世界の法則 - 存在強度](world/世界の法則.md#存在強度) |

## 3. 魂

| 概念 | 一言解説 | 出典 |
|---|---|---|
| **魂** | 真界から現界へマナを橋渡しする孔。個体ごとに異なる形を持つ | [世界の法則 - 魂](world/世界の法則.md#魂) |
| **魂の形（方向性）** | 通ったマナに方向性を与える。個体の個性にも影響（例：「熱い」魂→平熱が高い） | [世界の法則 - 魂](world/世界の法則.md#魂) |
| **魂の大きさ** | 一度に通せるマナの量を決定。成長過程でほぼ変わらない | [世界の法則 - 魂](world/世界の法則.md#魂) |
| **個体と魂の1対1原則** | 原則として1個体に1つの魂。ただし同調で他個体の魂にアクセス可能 | [世界の法則 - 魂](world/世界の法則.md#魂) |
| **同調** | 自分の精神エネルギーを伸ばして他個体の魂にアクセスすること。波長の近さ＋物理的距離の近さが必要 | [世界の法則 - 魂](world/世界の法則.md#魂) |

## 4. 精神エネルギー

| 概念 | 一言解説 | 出典 |
|---|---|---|

| **個体の定義** | 細胞→人間→地球→銀河など、視点によって異なるスケールの存在が各々精神エネルギーを持つ | [世界の法則 - 精神エネルギー](world/世界の法則.md#精神エネルギー) |
| **入れ子構造** | 地球の精神エネルギーは全生物の総和とは別に独立して存在する | [世界の法則 - 精神エネルギー](world/世界の法則.md#精神エネルギー) |
| **精神エネルギーの3属性** | 波長・強さ・方向の3つ | [世界の法則 - 精神エネルギー](world/世界の法則.md#精神エネルギー) |
| **波長** | 現界〜真界の2極間のグラデーション。真界寄りほど少量でマナと大量に共鳴可能 | [世界の法則 - 精神エネルギーの波長](world/世界の法則.md#精神エネルギーの波長) |
| **ルーツの近さと波長** | 宇宙誕生のルーツに近いほど真界寄り。進化するほど現界寄りになる | [世界の法則 - 精神エネルギーの波長](world/世界の法則.md#精神エネルギーの波長) |
| **強さ** | 意思の強さと物理的大きさに影響される。共鳴できるマナの量に影響 | [世界の法則 - 精神エネルギーの強さ](world/世界の法則.md#精神エネルギーの強さ) |
| **方向** | マナの変換先（どんな物理エネルギー/物質に変わるか）を決定。シンプルなほど変換容易 | [世界の法則 - 精神エネルギーの方向](world/世界の法則.md#精神エネルギーの方向) |
| **精神エネルギーの到達範囲** | 自己認識範囲＋五感で捉えた空間。後者は訓練で半径1m程度 | [世界の法則 - 精神エネルギーが生み出される範囲](world/世界の法則.md#精神エネルギーが生み出される範囲) |

## 5. 魔法

| 概念 | 一言解説 | 出典 |
|---|---|---|
| **魔法** | 個体自身の精神エネルギーでマナまたは瘴気を操ること。効率は高いが複雑な操作は不向き | [魔法 - 魔法とは](world/魔法.md#魔法とは) |
| **現化魔法** | マナを物質/エネルギーに変換する魔法。物質化とエネルギー化に細分 | [魔法 - 現化魔法](world/魔法.md#現化魔法) |
| **真化魔法** | マナ還元で現象を起こす魔法。物質還元とエネルギー還元に細分 | [魔法 - 真化魔法](world/魔法.md#真化魔法) |
| **真化魔法の制限** | ビッグバン由来物質は「現界」の精神エネルギーで保持されており実質還元不可能 | [魔法 - 真化魔法の制限](world/魔法.md#真化魔法の制限) |
| **魔法の暴走** | 精神エネルギーの制御喪失による意図しない現象。マナの暴走（自然現象）とは別 | [魔法 - 魔法の暴走](world/魔法.md#魔法の暴走) |
| **転送（ワープ）** | 真化＋現化の複合魔法。魂の孔が複数ある個体が前提。「消滅」でなく「移動」なので存在強度の制限に抵触しない | [魔法 - 転送（ワープ）](world/魔法.md#転送ワープ) |

## 6. 魔術

| 概念 | 一言解説 | 出典 |
|---|---|---|
| **魔術** | 術式で精神エネルギーを操作・変換し、マナを操ること。誰でも使えるが効率は魔法に劣る | [魔術 - 魔術とは](world/魔術.md#魔術とは) |
| **術式** | 精神エネルギーの波長をシフトするプログラム。波長シフトには指数関数的にエネルギー消費が増大 | [魔術 - 術式とは](world/魔術.md#術式とは) |
| **道具の魂を使用** | 魔術は基本的に術式が刻まれた道具の魂を経由してマナを取り出す（使用者の魂ではない） | [魔術 - 魔術とは](world/魔術.md#魔術とは) |
| **精神エネルギーを発生させる道具** | ケースA（集合無意識受信）、ケースB（封入）、ケースC（意思を持つ）の3パターン | [魔術 - 魔術の規模](world/魔術.md#魔術の規模) |
| **魔術の暴走** | 術式の欠陥・劣化・不適合・外的要因が原因。魔法の暴走とは原因が異なる | [魔術 - 魔術の暴走](world/魔術.md#魔術の暴走) |
| **ワープの魔術不可能性** | 真化＋現化の波長シフトが膨大になるため魔術ではほぼ不可能 | [魔術 - 転送（ワープ）](world/魔術.md#転送ワープ) |
