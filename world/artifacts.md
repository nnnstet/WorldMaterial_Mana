# 遺物・アーティファクト

## 遺物

- 神代・古代に存在した個体・道具・構造物のうち、その時代の精神エネルギーの痕跡を残して現存するもの
- 存在強度が極めて高いため、長い年月を経ても瘴気によるマナ還元に抵抗し、現界に残り続けている
  - 代行者・調停者が母体の魂を用いて作り出したもの（母体由来の存在強度）
  - 多くの個体に認知・使用されたことで集合無意識の裏付けを持つもの
  - 強い精神エネルギーが封入されたもの
- 遺物は道具に限らない
  - 構造物（神殿、要塞、儀式場など）
  - 空間そのもの（強い精神エネルギーが空間に刻まれたもの）
  - 残留する精神エネルギーの痕跡
- 遺物のうち術式が刻まれた道具で、道具自体が精神エネルギーを発生させるものは「伝説の武具・道具」として分類される（[後述](#伝説の武具道具)）
- 遺物の周辺では魔法環境が変質する傾向がある
  - 遺物に残る精神エネルギーが真界のマナとの共鳴を持続しており、周辺の魔法環境に影響を与える
  - この影響はダンジョン化の原因の一つとなる（詳細は[ダンジョン](dungeon.md)を参照）

## 種子

- 代行者・調停者が、自身の全滅を予見して残した「次世代の個体を発芽させるための遺物」
  - 代行者・調停者の精神エネルギーの痕跡と、母体（地球やエオス）の魂の方向性が封じ込められている
  - 特定の環境条件が満たされたときに発芽し、新たな個体・種族を生む
- 種子から発芽した個体は代行者・調停者の末裔としての性質を持つ
  - 母体の魂の方向性を引き継ぐ
  - 精神エネルギーの波長が代行者に近い傾向がある
- 発芽条件
  - 周辺の環境変化（対応する真界のマナの濃淡の変動、気候、生態系の変化など）
  - 地球自身の意思による促進も影響する
- 発芽した個体がテリトリーを構築し、集落を形成することがある
  - このテリトリーがダンジョン化するケースがある（詳細は[ダンジョン](dungeon.md#b-種子由来)を参照）
- 古代にはベローナの代行者の活動による環境変化と地球の意図的促進により、複数の種子が同時に発芽した（[年表](../history/timeline.md)参照）
  - エルフもエオスの代行者・調停者が残した種子から発芽した種族である
    - エオスの魂の方向性「成長・繁茂」が種子に封じ込められており、発芽した個体がこの方向性を引き継いでいる
- 古代において、ベローナの代行者の活動による環境変化と地球の意思による意図的促進により、神代に残された種子はすべて発芽済みである
  - 発芽条件が自然に整わなかった種子も、地球の意思により強制的に発芽させられた
  - 環境に適応できなかった個体・種族はそのまま滅亡した
  - 原則として、人代において新たな種子の発芽が起きることはない（ただし未知の特殊な事情による例外の可能性は排除しない）

## 伝説の武具・道具

### 概要

- 術式が刻まれた道具のうち、道具自体が精神エネルギーを発生させる特殊なもの
- 発生メカニズムの詳細は[魔術](魔術.md#魔術の規模)を参照
- 以下の精神エネルギー発生ケースに分類される（詳細は魔術.mdの「魔術の規模」を参照）
  - ケースA: 存在強度が極めて高い道具が集合無意識から精神エネルギーを受信
  - ケースB: 製作時の魔術師の精神エネルギーが封入されている
  - ケースC: 道具自体がある種の個体として意思を持つに至った

### 一覧

#### エクスカリバー

- **ケース**: A（集合無意識）
- **精神エネルギー源**: 「選定の剣」として万人に知られることによる極めて高い存在強度。集合無意識から精神エネルギーを受信する
- **特性**:
  - 使用者の資質（精神エネルギーの強さと方向性）に応じて出力が変動する
  - 使用者の精神エネルギーと剣が受信する集合無意識の精神エネルギーが相乗する構造
  - 「正統な王」としての認知が集合無意識を増幅するため、使用者の社会的立場も出力に影響する
- **術式**: 使用者の精神エネルギーを増幅する術式が刻まれており、集合無意識からの受信を使用者に還元する

#### 三種の神器（草薙剣）

- **ケース**: A + C
- **精神エネルギー源**: 国家レベルの集合無意識（「日本を守護するもの」としての認知）+ 「守護」という極限まで特化した意思
- **特性**:
  - 日本列島という大規模な個体（地球の一部）との同調を補助する
  - 使用者と日本列島の精神エネルギーの波長の差を術式が橋渡しすることで、通常は不可能な同調を実現する
  - 守護に関する魔法の効率が極めて高い一方、攻撃的な用途には適さない
- **術式**: 精神エネルギーの波長を日本列島の精神エネルギーに近づけるシフト機能を持つ

#### ミョルニル

- **ケース**: A + C
- **精神エネルギー源**: 「雷神の武器」としての集合無意識 + 「破壊」に極限まで特化した意思
- **特性**:
  - 現化魔法（エネルギー化: 電磁気エネルギー）に特化した武具
  - 術式が精神エネルギーの方向を「電磁気」に最適化するため、雷に関する魔法の効率が破格
  - 「破壊」の意思により、道具自体が精神エネルギーを発生させ、使用者の出力に加算される
- **術式**: 精神エネルギーの方向を電磁気エネルギーに変換する高効率な術式

#### ロンギヌスの槍

- **ケース**: C
- **精神エネルギー源**: 「貫く」という極限の意思。製作の経緯と歴史的な使用を通じて、この方向性が極限まで研ぎ澄まされた
- **特性**:
  - 真化魔法（還元）に特化した武具
  - 「貫く」という意思が存在強度を無視する方向で作用し、通常の真化魔法では不可能な高存在強度の対象に対しても還元を行える可能性がある
  - ただし「現界」の精神エネルギーを超えることは不可能であり、ビッグバン由来の物質への完全な還元はできない
- **術式**: 精神エネルギーをマナ還元の触媒として最適化する術式。対象の存在強度への干渉効率が極めて高い

#### 賢者の石

- **ケース**: B
- **精神エネルギー源**: 製作者の精神エネルギーが封入されている
- **特性**:
  - 使用者の魔法・魔術の出力をブーストする道具
  - 封入された精神エネルギーは使用により消耗するため、本質的には消耗品
  - ただし封入量が膨大であるため、実質的に数世代にわたって使用可能な場合もある
  - 意図的な製作は極めて困難であり、超希少
- **術式**: 封入された精神エネルギーを使用者の精神エネルギーに加算する術式
