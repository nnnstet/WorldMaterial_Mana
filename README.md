## 設計思想

本プロジェクトの世界設定は、6つの基本法則（公理）から各設定要素を論理的に導出する体系として構成されています。

- **公理**: 現界と真界の存在、マナ、個体、魂、精神エネルギー、マナの変換（現化・マナ還元）
- **定理**: 公理の組み合わせから導かれる法則（エネルギー保存則、瘴気、存在強度、マナ還元、魔法、魔術、超能力）
- **応用**: 定理に基づく世界の具体的な要素（遺物、ダンジョン、種族、歴史）

この構造により、魔法・魔術・超能力・種族・歴史といった多様な設定要素が、少数の前提から一貫した論理で説明されます。
新たな設定を追加する際にも、既存の公理・定理との整合性を確認することで、世界全体の一貫性を維持できます。

東アジアの「気」の概念、西洋的な二元論（真界/現界）、現代物理学の概念（エネルギー保存則、反物質）を統合した独自の体系です。
特定の文化や既存のファンタジー作品に依存するのではなく、「少数の前提から論理的に導出される体系」であることが本プロジェクトの特徴です。

導出構造の全体像は[導出構造マップ](world/README.md)を参照してください。
