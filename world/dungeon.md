# ダンジョン

## 概要

- ダンジョンとは、特定の原因により通常の自然環境とは異なる法則や生態系が形成された空間を指す
- 共通する特徴:
  - 内部の魔法環境が外部と異なる（マナとの共鳴のしやすさ、精神エネルギーの方向の偏りなどが変質している）
  - 独自の生態系が存在する（通常の環境では見られない個体が生息する）
  - 内部で独自のアイテム（武器、道具、素材など）が生成・存在する
- ダンジョンの規模はまちまちであり、小さな洞窟程度のものから大陸規模のものまでありうる
- ダンジョンの発生原因によって内部の特性が大きく異なるため、発生原因に基づいて分類される

## ダンジョンの共通メカニズム

### 魔法環境の変質

- ダンジョンの内部では、発生原因に起因する精神エネルギーの痕跡が真界のマナと持続的に共鳴している
- この持続的な共鳴が、以下のような形で魔法環境を通常と異なるものにする:
  - 特定の方向性の精神エネルギーが増幅される（遺物や暴走の方向性に引きずられる）
  - マナとの共鳴が促進または阻害される（その場所での魔法・魔術の効率が変動する）
  - 瘴気の残留量・分布が通常と異なる
  - 対応する真界領域のマナの濃淡に干渉し、通常の自然変動（数千〜数万年単位）を超える偏りを生じさせている場合もある
- 一般に「マナが濃い場所」と表現される現象は、上記のいずれか、または複合によるもの

### 瘴気の蓄積と影響

- ダンジョン内では、遺物・個体・暴走痕跡が持続的にマナを消費しているため、瘴気が蓄積する
- 蓄積した瘴気は周囲の最も存在強度が低いものから優先的に結合する（[瘴気の優先結合](世界の法則.md#マナ還元)）
- ダンジョンにおける瘴気の影響:
  - **物質化魔法への影響**: 蓄積した瘴気が生成物（存在強度が極めて低い）に集中結合し、通常より速く消滅する
  - **エネルギー化魔法（既存物質への作用）への影響**: 変化部分の存在強度が低い場合、蓄積した瘴気が優先的に結合し変化が速く戻る。強力な術者であれば変化部分の存在強度が高いため維持可能
  - **真化魔法への影響**: 瘴気が豊富であるため、還元に必要な瘴気の供給が潤沢で効率が上昇する
  - **現化魔法と真化魔法の非対称性**: 結果として、ダンジョン深部では「生み出す魔法」より「消す魔法」が相対的に有利になる
  - **瘴気の毒性**: 存在強度の低い個体は瘴気に体をむしばまれる。ダンジョンの奥ほど瘴気が蓄積しているため、弱い個体は深部に進入できない
- ダンジョンに長期間生息する個体は瘴気環境に適応しており、瘴気と共存する形で現化魔法を行使する能力を獲得している場合がある
  - これが攻略者との非対称性（ダンジョン個体は瘴気環境でも戦えるが、攻略者は制約を受ける）を生む

### アイテム生成の仕組み

ダンジョン内でアイテムが存在する理由を世界法則に基づいて説明する。分類によって仕組みが異なる。

#### 宝箱

| 分類 | 宝箱の正体 |
|---|---|
| A-1（魔物集結型） | 遺物の精神エネルギーが持続的にマナを変換し物質として安定させたもの |
| A-2（迷宮特性型） | 遺物の特性で生成された物質。遺物の方向性を反映 |
| B-1（テリトリー型） | 個体の保管品（収集・製作したもの） |
| C-1（魔物の巣型） | 魔物の保管品（他から略奪したもの含む） |
| C-2（暴走由来） | 暴走の痕跡に残る精神エネルギーがマナから持続生成した物質 |

#### ドロップアイテム

| 分類 | ドロップの正体 |
|---|---|
| A-1（魔物集結型） | 魔物が遺物の力を取り込んだ結果、身体や所持品に遺物の方向性が刻まれたもの。討伐時に物質として安定化 |
| A-2（迷宮特性型） | 遺物の方向性を反映した個体の素材 |
| B-1（テリトリー型） | 個体の所持品 |
| C-1（魔物の巣型） | 魔物の所持品 |
| C-2（暴走由来） | 暴走環境に適応した個体の身体素材、暴走で生成された物質 |

#### アイテム持ち出し後の安定性

- ダンジョンアイテムの安定性は[存在強度](世界の法則.md#存在強度)に依存する（受動的安定）
  - 遺物の精神エネルギーで十分な存在強度が付与され自律的に安定しているアイテムは、遺物から離れても即座に消滅しない
  - 長期間安定し多くの個体（ダンジョンの魔物含む）に認知されたアイテムほど存在強度が高く、持ち出し後も長期安定
  - 最近生成された低存在強度のアイテムは、持ち出し後は数日〜数週間で劣化する
- これがダンジョンアイテムのグレード差の法則的裏付けとなる

### ダンジョン内の生態系

- ダンジョン内の変質した魔法環境に適応した個体が生息する
- 長期間にわたる魔法環境の変質は、そこに生息する個体の進化や変異を促す
- ダンジョンの発生原因によって、主な生息個体が異なる（各分類で詳述）

## ダンジョンの分類

### A. 神代の遺物由来

#### A-1. 魔物集結型

- 神代の[遺物](世界の法則.md#遺物)が残る場所に、遺物の強い精神エネルギーの痕跡を感知した[魔物](../character/種族.md#魔物)が集まることで形成されるダンジョン
- **形成プロセス**:
  1. 神代の遺物が地中や遺跡に残存する
  2. 遺物の精神エネルギーの痕跡が真界のマナと持続的に共鳴し、周辺の魔法環境を変質させる
  3. 魔物が大きな力を本能的に感知し、集結する
  4. 集結した魔物が遺物の力を取り込み、さらに周辺の環境を変化させる
  5. この過程で独自の生態系と魔法環境が確立し、ダンジョン化する
- **内部の特性**:
  - 敵: 魔物が主体。遺物の力を取り込んだ個体は通常の魔物より強力
  - 遺物の方向性を反映した特性を持つ魔物が多い（例: 「熱」の遺物 → 火炎系魔物）
  - アイテム: 魔物が取り込んだ遺物の方向性が武器等の形で現れる（ドロップアイテムの仕組み）
  - 遺物本体も最深部等に存在する可能性がある
- **遺物の強さとダンジョンの難易度の関係**:
  - 遺物の精神エネルギーの痕跡が強いほど、感知範囲が広がり、より遠方からも魔物が集まる
  - 強い精神エネルギーの痕跡は、より強力な魔物を引きつける（魔物本来の個体としての強さが高い個体ほど、大きな力を感知・利用する能力に優れる）
  - 強い遺物ほど取り込める力も大きく、取り込んだ魔物がさらに強力になる
  - 結果として、遺物の強さがダンジョン全体の難易度・脅威度を決定する主要因となる
- **維持条件**: 遺物が存在する限り維持される。遺物除去で徐々に縮小するが、魔物が居座り続ける場合もある

#### A-2. 迷宮特性型

- 神代の遺物自体が「空間を構築・侵食する」方向性の精神エネルギーを持ち、周囲を侵食してダンジョン化したもの
- **形成プロセス**:
  1. 遺物の精神エネルギーの方向性が「構築」「拡張」「侵食」等の空間に作用するもの
  2. 遺物が真界のマナと持続的に共鳴し、周囲の空間を変質・拡張させる
  3. 長い年月をかけて大規模化する傾向がある
- **内部の特性**:
  - 迷宮そのものが遺物の精神エネルギーによって構築されている
  - **トラップ**: 遺物の方向性に基づくトラップが自動生成される。トラップがメインのケースもある
  - **敵: 2種類が敵対的に共存する**
    - 古代の魔物: A-1と同様に遺物の力に引き寄せられた魔物（ダンジョンにとっての部外者）
    - 遺物が生成した個体: 遺物の精神エネルギーが「構築」の延長として生み出した守護者的存在
    - この2種は互いに敵対関係（魔物は遺物の力を取り込もうとし、遺物生成個体はそれを排除しようとする）
  - **アイテム**: 遺物の特性で生成される（宝箱もドロップアイテムも遺物の方向性を反映）
- **維持条件**: 遺物が存在する限り迷宮は拡大し続ける（ただし極めて緩慢）

### B. 種子由来

#### B-1. テリトリー型

- [種子](世界の法則.md#種子)から発芽した個体がテリトリーを構築し、集落を形成することでダンジョン化したもの
- **形成プロセス**:
  1. 代行者・調停者が残した種子が発芽条件を満たし、個体が出現する
  2. 発芽した個体は代行者・調停者の末裔としての性質を持ち、周辺環境に精神エネルギーの影響を与える
  3. 個体がテリトリーを構築し、集落を形成する
  4. 集落の精神エネルギーが真界のマナとの共鳴に影響を与え、周辺の魔法環境を変質させてダンジョン化する
- **内部の特性**:
  - 集落としての構造を持つ（住居、通路、貯蔵庫、儀式場など）
  - 敵: 発芽した個体とその子孫
    - 知性を持つ場合もあり、その場合は侵入者に対して組織的に対応する
    - 必ずしも敵対的とは限らない（交渉可能な場合もある）
  - アイテム: **宝箱 = 保管品**（集落の個体が収集・製作したもの）、**ドロップ = 所持品**
  - 母体の魂の方向性を引き継いだ環境が特徴的
    - 例: エオス由来の種子 → 「成長・繁茂」 → 植物が異常に繁茂した森のダンジョン

### C. 古代の個体由来

#### C-1. 魔物の巣型

- 古代の魔物が巣を形成し、それがダンジョン化したもの
- **B-1（テリトリー型）と構造的に類似するが、個体が魔物である点が異なる**
- **形成プロセス**:
  1. 古代の魔物が集団で巣を形成する
  2. 魔物の「侵略」方向の精神エネルギーが周辺環境を侵食する
  3. 長い年月をかけて魔法環境が変質し、ダンジョン化する
- **内部の特性**:
  - B-1と同様に集落構造を持つが、「侵略」の方向性を反映して攻撃的・排他的
  - 敵: 魔物とその子孫。テリトリーへの侵入者に対して本能的に攻撃的
  - アイテム: **宝箱 = 保管品**（他のダンジョンや外部から略奪した物を含む）、**ドロップ = 所持品**
  - 「侵略」の方向性により、ダンジョンが周辺環境を侵食して拡大する傾向がある

#### C-2. 魔法・魔術の暴走由来

- 古代に行使された[魔法](魔法.md#魔法の暴走)や[魔術](魔術.md#魔術の暴走)の暴走の結果、その痕跡がダンジョン化したもの
- **形成プロセス**:
  1. 古代の大規模な魔法または魔術が暴走する
  2. 暴走により大量の精神エネルギーの痕跡が空間に刻まれる
  3. この痕跡が真界のマナと持続的に共鳴し、周辺の魔法環境を変質させ続ける
  4. 変質した魔法環境に適応した生態系が形成され、ダンジョン化する
- **内部の特性**:
  - 暴走した魔法・魔術の方向性がダンジョン全体の特性を決定する
    - 例: 熱エネルギーの魔法が暴走 → 異常な高温環境のダンジョン
    - 例: 物質化の魔法が暴走 → 意図しない物質が無秩序に生成され続ける空間
  - 環境そのものが危険（トラップというよりも環境ハザード）
  - 敵: 暴走の環境に適応した個体（通常の生物が変異したもの、または暴走で直接生成された個体）
  - アイテム: 暴走の精神エネルギーの痕跡がマナから持続的に物質を生成（宝箱相当）、暴走環境に適応した個体の身体素材（ドロップ）
- **維持条件**: 精神エネルギーの痕跡が消滅しない限り維持。痕跡は存在強度を持つため自然消滅に極めて長い時間がかかる

## 人代におけるダンジョン

古代のダンジョンは、古代末期の大変動（ベローナの代行者の自爆による環境激変・大量絶滅）および約2.5億年の地質変動（プレートテクトニクス）を経て、**ダンジョンとして認識可能な形では人代に残存していない**。

※ 遺物（道具類）が偶発的に出土することは既定（[魔法・魔術の歴史](../history/magic-history.md)参照）だが、それが「古代のダンジョンの核であった」と認識されることはない。ここでは、古代のダンジョンが人代でどのような状態にあるか、およびそれが現実の自然現象とどう対応するかを記述する。

### 各分類の残存状態と発見・認識されない理由

- **A-1（魔物集結型）**: ダンジョンとしては消滅
  - 遺物は存在強度の高さにより地中に残存するが、地質変動で地下深くに埋没している
  - 周囲に集まっていた魔物は古代末期の大変動で離散・死滅し、ダンジョンの生態系は完全に消失
  - 遺物が偶発的に出土した場合も、周辺にダンジョンを示す環境的な痕跡は残っていない
- **A-2（迷宮特性型）**: 遺物自体は存続する可能性があるが、人代からはアクセス不能
  - 2.5億年の地質変動で、遺物は現在の人類が到達できない地下深度に押し込まれている
  - 遺物が空間を能動的に維持していたとしても、地表との接点がなく発見手段がない
  - 仮に地下空洞として存在しても、通常の地質調査で捉えられる深度にはない
- **B-1（テリトリー型）**: ダンジョンとしては消滅
  - 古代末期の大変動で発芽個体が壊滅し、テリトリーの構造が失われた
- **C-1（魔物の巣型）**: 古代のものは消滅
  - 古代末期の大変動で魔物が滅亡し、巣の構造が失われた
- **C-2（暴走由来）**: 精神エネルギーの痕跡は残存しうるが、ダンジョンとしては認識不能
  - 2.5億年の歳月で精神エネルギーの痕跡は大幅に減衰
  - 残留する魔法環境の異常は、自然現象（地熱異常、鉱物分布の偏り、局所的な磁気異常など）として説明しうるレベルにまで弱まっている

### 人代の現実世界との対応

- 古代のダンジョンの痕跡が人代で検出可能な形で残存しないため、**人代の自然環境は現実世界と矛盾しない**
- 遺物（道具類）の出土は既定だが、これは考古学的な遺物の発見と同様の現象であり、「ダンジョン」という空間現象とは切り離されている
- 上記で唯一注意が必要なのは C-2（暴走由来）で、地熱異常や磁気異常として残留する可能性がある。ただしこれは現実世界にも地質学的原因で存在する現象であり、区別がつかない
